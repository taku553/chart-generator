# Chart Generator プロジェクト構造と各ファイルの役割

## プロジェクト構成（ツリー形式）

\`\`\`
chart-generator/
├── public/                              # 静的ファイル
│   ├── sample-data.csv                  # サンプルデータ（基本形式）
│   ├── sample-data-header-row2.csv      # サンプルデータ（複数行ヘッダー）
│   └── sample-good-data.csv             # サンプルデータ（推奨形式）
│
├── src/                                 # アプリケーションソースコード
│   ├── main.jsx                         # Reactアプリケーションのエントリーポイント
│   ├── App.jsx                          # ルートコンポーネント（アプリ全体の統括）
│   ├── App.css                          # グローバルスタイル、アニメーション定義
│   ├── index.css                        # Tailwind CSS設定とベーススタイル
│   │
│   ├── assets/                          # 画像などの静的アセット
│   │   └── react.svg                    # Reactロゴ
│   │
│   ├── components/                      # UIコンポーネント
│   │   ├── FileUpload.jsx               # ファイルアップロードと設定プロセス管理
│   │   ├── ChartDisplay.jsx             # グラフ表示と操作
│   │   ├── SheetSelector.jsx            # Excelシート選択
│   │   ├── DataRangeSelector.jsx        # データ領域選択
│   │   ├── SeparateHeaderSelector.jsx   # 列ヘッダー位置指定
│   │   ├── DataOrientationSelector.jsx  # データ向き確認
│   │   ├── HeaderRangeSelector.jsx      # ヘッダー行範囲選択
│   │   ├── DataLabelRangeSelector.jsx   # データ名列選択
│   │   ├── UnitSettings.jsx             # 単位とスケール設定
│   │   ├── ChartTitleSettings.jsx       # グラフタイトル設定
│   │   │
│   │   └── ui/                          # shadcn/ui ベースのUIコンポーネント
│   │       ├── (40以上のコンポーネント)
│   │
│   ├── lib/                             # ユーティリティ関数
│   │   ├── dataUtils.js                 # データ処理（解析、変換、型推定）
│   │   ├── dataTransform.js             # データ変換（転置、範囲抽出、結合）
│   │   ├── unitUtils.js                 # 単位とスケール処理
│   │   └── utils.js                     # 汎用ユーティリティ
│   │
│   └── hooks/                           # カスタムReactフック
│       └── use-mobile.js                # モバイル判定フック
│
├── ドキュメント/                         # プロジェクトドキュメント
├── dist/                                # ビルド出力ディレクトリ
├── node_modules/                        # 依存パッケージ
│
├── components.json                      # shadcn/ui 設定
├── package.json                         # プロジェクトメタデータ
├── vite.config.js                       # Vite設定
└── ... (その他設定ファイル)
\`\`\`

詳細は「project_design.md」を参照してください。
